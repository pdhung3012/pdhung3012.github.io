Research Interest

I am interested in Software Engineering (SE). My work focuses on proposing Machine Translation (MT) approaches for solving Software Engineering problems, including mining information from large scale source code repositories, constructing parallel corpuses from large scale source code repositories, applying and optimizing MT techniques for SE.
